1) Vérifiez votre version Python (le programme à été réalisé sous Python 3.10), il est préférable d'avoir une version égale
ou supérieur à la 3.10.

2) Afin de faire fonctionner le programme il faut installer la librairie mysql-connector-python, pour cela avec un terminal
ouvert dans le dossier ../ProjectManagment_Software/pyFiles/ executez la ligne :
        - pip3 install mysql-connector-python==8.0.29

3) Dans le but de tester toutes les fonctionnalités du programme il est important d'avoir accès à la base de donnée afin 
d'avoir une vue plus graphique des actions faîtes avec le programme. De plus cela sert de confirmation à l'action effectuée.
Pour cela :
        - rendez-vous sur cette page web : https://admin.alwaysdata.com/login/?next=/
        - rentrez le courriel et le mot de passe suivants : !!IDENTIFIANTS FOURNIS PAR MAIL!!
        - une fois sur le compte dirigez-vous sur la gauche de l'écran dans la colonne "Bases de données" et cliquez sur "MySQL"
        - cliquez ensuite sur "PhpMyAdmin" dans le petit encadré vers le haut du site
        - rentrez l'utilisateur MySQL et le mot de passe suivants : !!IDENTIFIANTS FOURNIS PAR MAIL!!
        - voilà vous avez accès à la base de donnée 
Si vous ne souhaitez pas vous connecter à la base de donnée au moment de l'execution du programme vous pouvez entrez 
l'utilisateur "Alex" pour vous connecter en tant qu'admin et pouvoir réaliser toutes les tâches subséquentes.

4) Enfin lors de l'execution du programme soyez sûr d'être connecté à internet (!!ATTENTION!! Edurom bloque la base de donnée)

5) Pour lancer le programme python nous avons utilisé l'execution de Visual Studio Code (Ctrl + F5) sur le fichier main.py.
En cas de problème lors de l'execution des étapes si dessus merci de contacter le responsable du projet Alex JURBERT.
